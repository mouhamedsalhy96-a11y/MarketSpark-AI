# MarketSpark AI 🚀

A modern, fast, AI-powered Micro-SaaS built to instantly generate high-converting marketing copy for e-commerce brands and solo founders.

## Features
- **Instant Copy Generation:** Turns a simple product description into multi-channel marketing assets in seconds.
- **Platform Specific:** Generates optimized content for LinkedIn, Twitter/X, and Email Newsletters.
- **1-Click Copy:** Streamlined UI with clipboard API integration for fast, frictionless workflows.
- **Secure Backend:** API keys and prompt logic are protected in secure Next.js server routes.

## Tech Stack
- **Frontend:** Next.js (App Router), React, Tailwind CSS, Lucide Icons
- **Backend:** Next.js Route Handlers
- **AI Integration:** Google Gemini API (gemini-2.5-flash)
- **Deployment:** Vercel

## Local Development
To run this project locally, add your Gemini API key to a `.env.local` file:
`GEMINI_API_KEY=your_key_here`
Then run `npm run dev`.
